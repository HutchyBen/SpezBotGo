# SpezBot
### A discord bot that plays music for me and copains

## Features
- Plays Music
- Uses slash commands
- Has a fair queue system
- Better written than C# version
- Spams your server with randomly generated text

## Commands
- `/echo`
- `/forcestop`
- `/kill`
- `/nowplaying`
- `/pause`
- `/play`
- `/playlist`
- `/playnext`
- `/queue`
- `/resume`
- `/seek`
- `/shuffle`
- `/skip`
- `/volume`
